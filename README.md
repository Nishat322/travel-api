## API Documentation

### Thoughtful Endpoints 

#### /api/thoughts
    GET: Return an array of total thoughts
    POST: Add a thought to the list

#### /api/thoughts/:thought_id
    GET: Return a thought
    DELETE: Delete a thought from the array
